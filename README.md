# Tower Defense em Java

Este projeto é um jogo do tipo Tower Defense desenvolvido em Java como projeto final.

## Sobre o jogo
O jogador deve defender sua base impedindo que os inimigos avancem pelo caminho.
Torres podem ser posicionadas estrategicamente para atacar os inimigos conforme a
distância entre eles e as torres.

Inicialmente, o jogo foi idealizado para rodar no terminal, com foco na lógica,
cálculo de distâncias e regras do jogo.

## Tecnologias utilizadas
- Java
- Programação Orientada a Objetos

## Objetivo do projeto
Aplicar conceitos de lógica de programação, estruturas de dados e POO em um projeto prático.
